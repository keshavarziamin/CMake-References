
# CMake Resources Guide

This repository is a comprehensive collection of resources to help developers at any level improve their understanding and use of CMake. Whether you're just starting out, or you're an experienced user looking to deepen your knowledge, this list covers the official documentation, community resources, books, videos, and real-world projects that showcase CMake in action.

### Websites

- **[CMake Official Page](https://cmake.org/)**  
  CMake's official website is the primary source for downloading the latest releases and accessing official documentation. It includes tutorials, examples, and FAQs, as well as links to other resources. A good first stop for beginners and reference for experienced users.

- **[CMake Community Wiki](https://gitlab.kitware.com/cmake/community/-/wikis/home)**  
  This community-maintained wiki offers a wide range of resources, including examples, best practices, and answers to frequently asked questions. It’s a great resource for finding unique use cases and community solutions to common problems.

- **[Wikipedia - CMake](https://en.wikipedia.org/wiki/CMake)**  
  A general overview of CMake, including its history, purpose, and technical details. Wikipedia also provides insight into the use of CMake across different platforms and languages, giving a contextual foundation for understanding its role in software development.

- **[Awesome CMake](https://github.com/onqtam/awesome-cmake)**  
  This is a curated GitHub repository that lists valuable CMake resources, including scripts, modules, tutorials, and example projects. It’s maintained by the open-source community, making it a dynamic source of ideas and inspiration for practical CMake usage.

### Books

- **[CMake: An Introduction (Free Document)](https://drive.google.com/file/d/1dojiLWIKvY_OlhJXeEoqiYvNf3O8c_7v/view?usp=drive_link)**  
  A beginner-friendly guide available directly from CMake’s official documentation. This resource walks new users through basic concepts and builds up to more complex configurations, making it ideal for getting started with CMake.

- **[Modern CMake - A Free Book Created by GitLab](https://drive.google.com/file/d/1WWjuPQHA3O1QL0JEXiTYOFW_lfDFRFMR/view?usp=drive_link)**  
  This is a free and frequently updated book that teaches current best practices for using CMake in modern C++ projects. Topics range from basic setup to advanced configuration, and it’s particularly valuable for those looking to work with modern C++ standards.

- **[Mastering CMake](https://drive.google.com/file/d/1XSEYqD6st9xYXjqYsK5_yk1dkFBm9csx/view?usp=drive_link)**  
  Written by Kitware, the creators of CMake, this comprehensive book covers in-depth details on CMake’s functionality. It’s a valuable resource for understanding the nuances of CMake and can be especially useful for advanced users and developers working on complex projects.

- **[Learning CMake (Stack Overflow eBook)](https://drive.google.com/file/d/1LQ5PCJvn4r7Lgatmw61Qf4eq9hkVPs2r/view?usp=drive_link)**  
  Created from Stack Overflow contributions, this eBook compiles common questions, solutions, and best practices related to CMake. It’s ideal for troubleshooting and understanding real-world scenarios developers face.

- **[Professional CMake: A Practical Guide](https://crascit.com/professional-cmake/)**  
  Widely regarded as the ultimate handbook for CMake users, this book covers CMake comprehensively, from basics to advanced topics. It includes real-world examples and best practices gathered from production projects, making it an excellent resource for all CMake users.

### Practical Projects Using CMake

- **[LoRaMac-node](https://github.com/Lora-net/LoRaMac-node)**  
  An open-source implementation of a LoRaWAN end-device stack, this project demonstrates how to use CMake in an embedded environment. The codebase offers a good example of structuring a CMake project for embedded devices and managing dependencies.

- **[OpenCV](https://github.com/opencv/opencv)**  
  The widely-used computer vision library OpenCV uses CMake to handle its complex multi-platform build requirements. Exploring OpenCV’s CMake files provides insight into managing extensive dependencies and configurations in cross-platform projects.


### How to Use This List

- **Beginners**: Start with the [CMake Official Page](https://cmake.org/) and [CMake: An Introduction](https://drive.google.com/file/d/1dojiLWIKvY_OlhJXeEoqiYvNf3O8c_7v/view?usp=drive_link) book for foundational knowledge.
- **Intermediate Users**: Explore the community wiki and books like *Modern CMake (- A Free Book Created by GitLab)* to enhance your workflow.
- **Advanced Users**: Refer to *Professional CMake: A Practical Guide* and practical projects such as [LoraMac](https://github.com/Lora-net/LoRaMac-node) to understand best practices in large-scale applications.
