CMake Resources Guide

This repository is a comprehensive collection of resources to help developers at any level improve their understanding and use of CMake. Whether you're just starting out, or you're an experienced user looking to deepen your knowledge, this list covers the official documentation, community resources, books, videos, and real-world projects that showcase CMake in action.

Table of Contents

  • Websites
  • Books
  • Videos
  • Practical Projects Using CMake
  • How to Use This List

Websites

  • CMake Official Page: Primary source for downloading the latest releases and accessing official documentation.
  • CMake Community Wiki: A community-maintained wiki with resources, examples, and best practices.
  • Wikipedia - CMake: General overview and historical background of CMake.
  • Awesome CMake: Curated list of valuable CMake resources.

Books

  • CMake: An Introduction: Beginner-friendly guide with basic concepts and examples.
  • Modern CMake: A free book teaching current best practices.
  • Mastering CMake: Comprehensive guide from Kitware, ideal for advanced users.
  • Learning CMake (Stack Overflow eBook): Compilation of common questions and solutions.
  • Professional CMake: A Practical Guide: Widely regarded as the ultimate handbook for CMake users.

Videos

  • CMake Tutorial - Code, Tech, and Tutorials Channel: Beginner-friendly tutorial series.
  • Better CMake by Jefferson Amstutz: Modern CMake techniques and best practices.
  • CMake, Tests, and Tooling for C/C++ Projects: In-depth Udemy course on testing and CI/CD pipelines.

Practical Projects Using CMake

  • LoRaMac-node: Example of using CMake in an embedded environment.
  • OpenCV: Demonstrates handling multi-platform builds with CMake.

How to Use This List

  • Beginners: Start with the official page and the CMake: An Introduction book.
  • Intermediate Users: Explore the community wiki and books like Modern CMake.
  • Advanced Users: Refer to Professional CMake and explore practical projects like LoRaMac-node.
